# 📝 Full-Stack Blog Web Application

A simple and functional full-stack blog platform that allows users to create, edit, and manage posts with full CRUD capabilities without the persistance of data. Built using an Express backend and EJS templating, this project demonstrates core web development concepts and server-side rendering.

## 🌐 Live Demo

Check out the deployed app here:
👉 https://blog-web-application-mhip.onrender.com/

---

## 🚀 Features

* ✍️ Create new blog posts
* 📖 View all published posts
* 🛠️ Edit existing posts
* 🗑️ Delete posts
* 🧩 Server-side rendering with EJS templates

---

## 🧱 Tech Stack

**Frontend**

* EJS (Embedded JavaScript Templates)
* CSS3

**Backend**

* Node.js
* Express.js

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/blog-web-application.git
```

2. Navigate into the project directory:

```bash
cd blog-web-application
```

3. Install dependencies:

```bash
npm install
```

4. Start the server:

```bash
node index.js
```

5. Open your browser and go to:

```
http://localhost:3000
```

---

## 🧠 How It Works

* The application uses **Express.js** to handle routing and server logic.
* **EJS templates** render dynamic content on the server side.
* CRUD operations allow users to:

  * Create posts via forms
  * Read/display posts on the homepage
  * Update posts through edit forms
  * Delete posts from the list
